# 系统详细测试用例清单（毕业设计参考）



## 2. 音乐资源管理模块测试

| 测试点ID | 测试功能 | 测试步骤描述 | 预期结果 |
| :--- | :--- | :--- | :--- |
| **TC-RES-01** | **曲库扫描-路径配置** | 在管理员后台设置有效的本地音乐目录路径。 | 配置保存成功，后端确认路径可访问。 |
| **TC-RES-02** | **曲库扫描-执行扫描** | 点击“立即扫描”按钮。 | 扫描进度条显示（如有），后端日志显示遍历过程，扫描结束后提示新增/更新数量。 |
| **TC-RES-03** | **元数据解析** | 检查扫描后的歌曲列表。 | 歌曲的标题、歌手、专辑、时长、年份等信息与文件 ID3 标签一致。 |
| **TC-RES-04** | **封面提取与去重** | 扫描包含内嵌封面的音频文件。 | 封面图片被提取至 `data/cover` 目录，相同封面的不同歌曲指向同一张封面图片（哈希去重）。 |
| **TC-RES-05** | **资源检索** | 在搜索框输入歌曲名、歌手或专辑关键词。 | 列表实时/回车后显示匹配结果，支持模糊查询。 |

## 3. 播放控制模块测试

| 测试点ID | 测试功能 | 测试步骤描述 | 预期结果 |
| :--- | :--- | :--- | :--- |
| **TC-PLAY-01** | **基础播放** | 点击歌曲列表中的“播放”按钮。 | 底部播放器加载歌曲，封面、歌名显示正确，音频开始播放，播放按钮变为暂停状态。 |
| **TC-PLAY-02** | **播放控制** | 点击暂停、上一首、下一首按钮。 | 音频暂停/恢复，切歌逻辑符合当前播放列表顺序。 |
| **TC-PLAY-03** | **进度跳转** | 拖拽进度条到指定位置。 | 音频跳转到对应时间点继续播放，歌词同步跳转。 |
| **TC-PLAY-04** | **播放模式切换** | 点击播放模式图标（顺序、随机、单曲循环）。 | 图标状态变更，切歌逻辑随之改变（如单曲循环结束后重播当前歌曲）。 |
| **TC-PLAY-05** | **歌词同步** | 播放带有歌词的歌曲。 | 歌词面板随进度滚动，当前句高亮显示，无歌词时显示默认提示。 |
| **TC-PLAY-06** | **音量控制** | 拖拽音量条或点击静音图标。 | 实际输出音量变化，静音时图标变为静音状态。 |
| **TC-PLAY-07** | **状态持久化** | 刷新页面。 | 页面重新加载后，播放器恢复刷新前的歌曲、进度、音量和播放列表。 |

## 4. 歌单与互动模块测试

| 测试点ID | 测试功能 | 测试步骤描述 | 预期结果 |
| :--- | :--- | :--- | :--- |
| **TC-LIST-01** | **创建私有歌单** | 点击“新建歌单”，输入标题和描述。 | 歌单创建成功，左侧导航栏出现新歌单。 |
| **TC-LIST-02** | **添加歌曲** | 在歌曲列表右键或点击菜单选择“添加到歌单”。 | 歌曲成功加入目标歌单，歌单歌曲数量 +1。 |
| **TC-LIST-03** | **移除歌曲** | 在歌单详情页移除某首歌曲。 | 歌曲从歌单中消失，但不删除库中歌曲文件。 |
| **TC-LIST-04** | **公共歌单订阅** | 浏览公共歌单广场，点击“收藏/订阅”。 | 该歌单出现在用户的“收藏歌单”列表中。 |
| **TC-LIST-05** | **歌曲点赞** | 点击播放器或列表中的“红心”图标。 | 图标变为红色实心，歌曲加入“我喜欢的音乐”列表；再次点击取消喜欢。 |
| **TC-LIST-06** | **播放历史记录** | 完整播放一首歌曲。 | “最近播放”列表中出现该歌曲记录，且时间更新为最新。 |

## 6. AI 增强功能测试

| 测试点ID | 测试功能 | 测试步骤描述 | 预期结果 |
| :--- | :--- | :--- | :--- |
| **TC-AI-01** | **AI 描述生成** | 针对某位歌手点击“生成 AI 介绍”。 | 前端显示 Loading，后端调用大模型 API，数秒后歌手详情页显示生成的文本介绍。 |

| **TC-AI-03** | **播客模式播放** | 开启“播客模式”并播放该歌曲。 | 播放器先播放开场白音频（TTS），结束后无缝衔接播放歌曲正文。 |
