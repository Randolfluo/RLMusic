# AI Prompts

This directory contains markdown templates used for AI generation tasks.

## Files

- **prompt_playlist1.md**: Prompt for analyzing playlist content (Step 1).
- **prompt_playlist2.md**: Prompt for generating playlist description based on analysis (Step 2).
- **prompt_artist1.md**: Prompt for analyzing artist style/bio (Step 1).
- **prompt_artist2.md**: Prompt for generating artist description (Step 2).
- **prompt_album1.md**: Prompt for analyzing album concept (Step 1).
- **prompt_album2.md**: Prompt for generating album description (Step 2).
- **prompt_podcast1.md**: Prompt for analyzing podcast content.
- **prompt_podcast2.md**: Prompt for generating podcast description.
- **prompt_podcast3.md**: Prompt for podcast summarization.

## Usage

These files are read by the server application to construct prompts for the LLM.
